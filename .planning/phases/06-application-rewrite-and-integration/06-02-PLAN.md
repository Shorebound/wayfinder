    **Header (SDLRenderDeviceSubsystem.h):**
    - `class SDLRenderDeviceSubsystem final : public AppSubsystem` in `Wayfinder` namespace
    - Public methods from SDLGPUDevice: `GetDevice() -> SDL_GPUDevice*`, `GetSwapchainFormat()`, `CreateBuffer(...)`, `CreateTexture(...)`, `SubmitCommandBuffer(...)`, plus whatever GPU resource methods SDLGPUDevice exposes
    - `Initialise(EngineContext& context) -> Result<void>` - gets SDLWindowSubsystem from context via `context.GetAppSubsystem<SDLWindowSubsystem>()`, calls SDL_CreateGPUDevice, claims window for swapchain via SDL_ClaimWindowForGPUDevice

    - `BeginFrame() / EndFrame()` methods if SDLGPUDevice has frame lifecycle
    - Keep the RenderDevice public interface that other subsystems need (like Renderer)

    **Implementation (SDLRenderDeviceSubsystem.cpp):**
    - Merge SDLGPUDevice::Initialise() + RenderDeviceSubsystem::Initialise() flows
    - Access window via `context.GetAppSubsystem<SDLWindowSubsystem>().GetNativeWindow()`
    - Read config from ConfigService if needed (check existing RenderDeviceSubsystem.cpp for config usage)

    **Important:** The existing `RenderDevice` abstract base class in `rendering/backend/` may be referenced by the Renderer class. If so, keep RenderDevice abstract class temporarily -- SDLRenderDeviceSubsystem can also inherit from it (multiple inheritance: `public AppSubsystem, public RenderDevice`), or Renderer can be updated to use SDLRenderDeviceSubsystem directly. Read the Renderer's interface needs before deciding.

    **Do NOT:**
    - Reference BackendConfig.h or RenderBackend enum
    - Create null RenderDevice variant (D-02)

    **SDLPlatformPlugins.h/.cpp - contains 4 types:**

    1. `SDLWindowPlugin : public IPlugin` - Describe returns `{.Name = "SDLWindowPlugin"}`, Build calls `builder.RegisterAppSubsystem<SDLWindowSubsystem>({.RequiredCapabilities = {Capability::Presentation}})`.
    2. `SDLInputPlugin : public IPlugin` - Describe returns `{.Name = "SDLInputPlugin"}`, Build calls `builder.RegisterAppSubsystem<SDLInputSubsystem>()` (no RequiredCapabilities - always active).
    3. `SDLTimePlugin : public IPlugin` - Describe returns `{.Name = "SDLTimePlugin"}`, Build calls `builder.RegisterAppSubsystem<SDLTimeSubsystem>()` (no RequiredCapabilities - always active).
    4. `struct SDLPlatformPlugins` - PluginGroup (NOT an IPlugin). `Build(AppBuilder& builder)` calls `builder.AddPlugin<SDLWindowPlugin>(); builder.AddPlugin<SDLInputPlugin>(); builder.AddPlugin<SDLTimePlugin>();`

    - `SDLRenderDevicePlugin : public IPlugin`
    - Describe: `{.Name = "SDLRenderDevicePlugin"}`
    - Build: `builder.RegisterAppSubsystem<SDLRenderDeviceSubsystem>({.RequiredCapabilities = {Capability::Rendering}, .DependsOn = Deps<SDLWindowSubsystem>()});`
