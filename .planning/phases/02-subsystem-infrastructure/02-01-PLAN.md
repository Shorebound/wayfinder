**SubsystemRegistry<TBase> template class:**

Private storage:
```cpp
struct Entry
{
    std::type_index ConcreteType;
    std::optional<std::type_index> AbstractType;
    SubsystemDescriptor Descriptor;
    std::function<std::unique_ptr<TBase>()> Factory;
    std::unique_ptr<TBase> Instance;
    std::string DebugName;
    bool Active = false;
};

std::vector<Entry> m_entries;
std::unordered_map<std::type_index, size_t> m_typeToIndex;
std::unordered_map<std::type_index, std::type_index> m_abstractRedirect;
std::vector<size_t> m_initOrder;
bool m_finalised = false;
```

**Public API (all methods use trailing return types, `and`/`or`/`not`, west const, British spelling):**

`Register<T>(SubsystemDescriptor descriptor = {})`:
- `requires std::derived_from<T, TBase>`
- `WAYFINDER_ASSERT(not m_finalised, "Cannot register after Finalise()")`
- Check duplicate via `m_typeToIndex.contains(typeid(T))` - assert if duplicate
- Push Entry with ConcreteType=typeid(T), Factory=`[]{ return std::make_unique<T>(); }`, DebugName from demangled or raw `typeid(T).name()`
- Insert into m_typeToIndex

`Register<TConcrete, TAbstract>(SubsystemDescriptor descriptor = {})`:
- `requires (std::derived_from<TConcrete, TBase> and std::derived_from<TConcrete, TAbstract>)`
- Same as above PLUS set AbstractType=typeid(TAbstract)
- Insert `m_abstractRedirect[typeid(TAbstract)] = typeid(TConcrete)`
- WAYFINDER_ASSERT no duplicate abstract (reject if m_abstractRedirect already has typeid(TAbstract))

`Finalise() -> Result<void>`:
1. Validate all DependsOn targets exist in m_typeToIndex:
   - For each entry, for each dep in entry.Descriptor.DependsOn: if `not m_typeToIndex.contains(dep)` return `MakeError(std::format("Subsystem '{}' declares dependency on unregistered type", entry.DebugName))`
2. Validate abstract redirect targets exist (each value in m_abstractRedirect must be in m_typeToIndex)
3. Build adjacency list: `std::vector<std::vector<size_t>> adjacency(m_entries.size())` where adjacency[depIdx] contains entryIdx (dep -> dependent)
4. Compute in-degree for each entry
5. Run Kahn's algorithm:
   - Seed queue with all entries having 0 in-degree (use a queue, tiebreak by index for determinism)
   - Dequeue, append to m_initOrder, decrement dependents' in-degree, enqueue new zero-in-degree
6. If `m_initOrder.size() != m_entries.size()`:
   - Extract cycle path: from any node with non-zero in-degree, DFS following only residual edges (to other non-zero-in-degree nodes). When revisiting a node, extract the cycle between the two visits.
   - Format: `"Cycle detected: A -> B -> C -> A"`
   - Return `MakeError(cycleErrorMessage)`
7. Set `m_finalised = true`
8. Return `{}`

`Initialise(EngineContext& context, const CapabilitySet& effectiveCaps) -> Result<void>` (per D-06, D-17, D-19, CAP-03, CAP-04):
1. `WAYFINDER_ASSERT(m_finalised, "Must call Finalise() before Initialise()")`
2. For each index `i` in m_initOrder:
   a. `auto& entry = m_entries[i]`
   b. Capability check: `if (not entry.Descriptor.RequiredCapabilities.IsEmpty() and not effectiveCaps.HasAll(entry.Descriptor.RequiredCapabilities))` then `continue` (CAP-03, CAP-04: empty = always active)
   c. `entry.Instance = entry.Factory()`
   d. `auto result = entry.Instance->Initialise(context)` -- calls the v2 virtual on TBase
   e. If `not result`:
      - Reverse-shutdown: iterate m_initOrder backwards from current position, for each with Active instance call Shutdown() and reset
      - Return `std::unexpected(result.error())`
   f. `entry.Active = true`
3. Build lookup: for each active entry, insert pointer into a lookup map for Get/TryGet
4. Return `{}`

`Shutdown()` (per D-07):
- Iterate m_initOrder in REVERSE. For each entry with `Active` and `Instance`, call `Instance->Shutdown()`, set Active=false, reset Instance.
- Clear lookup maps.

`Get<T>() -> T&` (per D-08):
- Resolve type_index: check m_typeToIndex directly; if not found check m_abstractRedirect to get concrete type, then look up concrete
- WAYFINDER_ASSERT the entry exists and is Active
- Return `static_cast<T&>(*entry.Instance)`

`TryGet<T>() -> T*`:
- Same resolution as Get but return nullptr if not found or not Active

Const overloads for both Get and TryGet returning `const T&` / `const T*`.

`IsRegistered<T>() const -> bool` (utility).

`IsFinalised() const -> bool` (utility).
