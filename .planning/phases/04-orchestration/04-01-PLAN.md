    2. **SuspensionPolicy** struct:
       - `bool AllowBackgroundUpdate = false;`
       - `bool AllowBackgroundRender = true;`
       Default: foreground allows background to render (common pause-over-gameplay case) but not update.

       - `bool Render;`

       ```cpp
       [[nodiscard]] constexpr auto ComputeBackgroundPolicy(
           const BackgroundPreferences& background,
           const SuspensionPolicy& foreground) -> EffectiveBackgroundPolicy
       {
           return {
               .Update = background.WantsBackgroundUpdate and foreground.AllowBackgroundUpdate,
               .Render = background.WantsBackgroundRender and foreground.AllowBackgroundRender,
           };
       }
       ```

    5. **PendingOperation types** (per research Pattern 1):
       ```cpp
       struct FlatTransition { std::type_index Target; };
       struct PushTransition { std::type_index Target; };
       struct PopTransition {};

       ```
       Include `<typeindex>` and `<variant>`.

    6. **StateRegistrationDescriptor** struct:
       - `std::function<std::unique_ptr<IApplicationState>()> Factory;`
       - `CapabilitySet Capabilities;`
