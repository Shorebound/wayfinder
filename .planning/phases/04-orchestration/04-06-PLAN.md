Provides `CapabilitySet` containing Simulation+Rendering via a `GetCapabilities()` test helper (not IApplicationState method - stored in state manifest).

    `IntegMockStateUI`: Derives from IStateUI. Has AttachCalled/DetachCalled/SuspendCalled/ResumeCalled/UpdateCalled flags. OnAttach/OnDetach return success. GetName "IntegStateUI".

    Helper `SetupCapabilities()`: Creates TagRegistry, calls NativeTag::RegisterAll, returns registry. Needed to make Capability::Rendering.GetTag() valid.

    Helper `BuildOrchestration()`: Creates an ApplicationStateMachine from a StateManifest, an OverlayStack from an OverlayManifest, wires into EngineContext. Returns a struct with all the pieces (ASM, OverlayStack, EngineContext, owned overlays vector, lifecycle hook manifest) so tests can inspect them.

    **TEST_SUITE("Orchestration Integration"):**

    1. **"EngineContext.RequestTransition delegates to ASM"** - Build orchestration with states A->B. Start A. ctx.RequestTransition<IntegMockStateB>(). asm.ProcessPending(ctx). Active state type is IntegMockStateB.

    2. **"EngineContext.RequestPush delegates to ASM"** - Build with A + pushable B. Start A. ctx.RequestPush<IntegMockStateB>(). asm.ProcessPending(ctx). Stack size == 2.

    3. **"EngineContext.RequestPop delegates to ASM"** - Push B over A. ctx.RequestPop(). asm.ProcessPending(ctx). Stack size == 1. Active is A.

    4. **"EngineContext.ActivateOverlay delegates to OverlayStack"** - Add overlay, deactivate it. ctx.ActivateOverlay(typeid(IntegMockOverlay)). Overlay IsActive() == true.

    5. **"EngineContext.DeactivateOverlay delegates to OverlayStack"** - Active overlay. ctx.DeactivateOverlay(typeid(IntegMockOverlay)). Overlay IsActive() == false.

    6. **"Full transition cascade: state change triggers overlay activation"** - State A provides Rendering capability. Overlay requires Rendering. Start A. Overlay should be active (OnAttach called). Transition to state B (which does NOT provide Rendering). After ProcessPending: overlay should be inactive (OnDetach called). This is the core cascade test.

    7. **"IStateUI attaches on state enter and detaches on state exit (UI-03)"** - Register IntegMockStateUI for state A. Start A. IntegMockStateUI.AttachCalled == true. Transition to B. IntegMockStateUI.DetachCalled == true.

    8. **"IStateUI suspends/resumes on push/pop (UI-03)"** - Register IntegMockStateUI for state A. Start A. Push B. IntegMockStateUI.SuspendCalled == true. Pop. IntegMockStateUI.ResumeCalled == true.

    9. **"Lifecycle hooks fire during transitions"** - Register OnStateEnter<IntegMockStateB> hook via LifecycleHookManifest. Start A. Transition to B. ProcessPending. Hook fired (bool flag set by hook lambda).

    10. **"Overlay event consumption blocks state event processing"** - Active overlay with ConsumeEvents = true. ProcessEvents on OverlayStack returns true. State's OnEvent should NOT be called (simulated by checking return value signals consumption).
