    **StateMachine.h** (per D-08, D-09, D-10, D-11):

    Header-only template in `namespace Wayfinder`. Include `core/Result.h`, `<cassert>`, `<functional>`, `<optional>`, `<queue>`, `<unordered_map>`, `<unordered_set>`, `<vector>`, `<format>`.

    `StateDescriptor<TStateId>` struct:
    - `TStateId Id`
    - `std::function<void()> OnEnter`
    - `std::function<void()> OnExit`
    - `std::vector<TStateId> AllowedTransitions`

    `StateMachine<TStateId>` class with `requires std::equality_comparable<TStateId>`:
    - **Registration (pre-Finalise):**
      - `void AddState(StateDescriptor<TStateId> descriptor)` - stores descriptor in `m_states` map keyed by Id
    - **Validation:**
      - `[[nodiscard]] auto Finalise(const TStateId& initialState) -> Result<void>`:
        1. Check `initialState` is registered, else return `MakeError("Initial state not registered")`
        2. Check every AllowedTransitions target is registered, else return `MakeError` with descriptive message
        3. BFS from `initialState` through AllowedTransitions; any registered state not visited = unreachable, return `MakeError`
        4. Check for duplicate registrations (handled at AddState time or here)
        5. On success: store `m_initialState`, set `m_finalised = true`, return `{}`
    - **Runtime:**
      - `void Start()` - assert `m_finalised`, set `m_currentState = m_initialState`, call OnEnter, set `m_running = true`
      - `void TransitionTo(const TStateId& target)` - assert `m_running`, check target is in current state's AllowedTransitions (assert if not), store as `m_pendingTransition`
      - `void ProcessPending()` - if `m_pendingTransition` has value: call current OnExit, update `m_previousState = m_currentState`, update `m_currentState = target`, call new OnEnter, fire observers, clear pending
      - `[[nodiscard]] auto GetCurrentState() const -> const TStateId&`
      - `[[nodiscard]] auto GetPreviousState() const -> std::optional<TStateId>` (nullopt if no transition yet)
      - `[[nodiscard]] auto IsRunning() const -> bool`
      - `void OnTransition(TransitionCallback cb)` where `TransitionCallback = std::function<void(const TStateId&, const TStateId&)>` - registers observer
    - **Storage:**
      - `std::unordered_map<TStateId, StateDescriptor<TStateId>, Hash>` m_states (use `typename TStateId::Hash` if available, else `std::hash<TStateId>`)
      - Use a template parameter or helper for the hash. Simplest: require `std::hash<TStateId>` works (InternedString provides it via `InternedString::Hash`; for enum class, a Lambda-based hash or `std::hash<std::underlying_type_t<TStateId>>` wrapper).
      - Actually, use a second template parameter with default: `template<typename TStateId, typename THash = std::hash<TStateId>>`
    - **@todo comment:** "Unlock()/Revalidate() escape hatch for hot-reload/mod support" per D-10
