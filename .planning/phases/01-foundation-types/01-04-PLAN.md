<objective>
Remove the dead-end DLL plugin system: delete PluginExport.h, PluginLoader.h/.cpp, and EntryPoint.h. Update all consumers (journey, waystone, waypoint) to use direct plugin construction with explicit main() functions.

Purpose: The DLL plugin system was a speculative feature that adds complexity without benefit. The engine uses static linking. Removing it simplifies the build, reduces attack surface, and removes macro-based entry points.
Output: 4 engine files deleted, 3 consumer files updated, CMakeLists.txt cleaned, AGENTS.md updated.
</objective>
