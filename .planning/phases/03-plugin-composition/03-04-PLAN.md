**1. Create `engine/wayfinder/src/app/ConfigRegistrar.h`** in `namespace Wayfinder`:

ConfigRegistrar tracks config types declared during Build phase. Derives from IRegistrar for store compatibility.

```cpp
class ConfigRegistrar : public IRegistrar
{
public:
    /// Declare a config type associated with a TOML file key.
    /// Called by AppBuilder::LoadConfig<T>(key). Tracks the key for validation.
    void DeclareConfig(std::string_view key, std::type_index type);

    /// Load and cache a TOML table for the given key.
    /// Uses 3-tier layering: config/<key>.toml merged with saved/config/<key>.toml.
    /// Returns the merged table (cached on subsequent calls for the same key).
    [[nodiscard]] auto LoadTable(std::string_view key,
        const std::filesystem::path& configDir,
        const std::filesystem::path& savedDir) -> Result<const toml::table&>;

    struct ConfigEntry
    {
        std::string Key;
        std::type_index Type;
    };

    [[nodiscard]] auto GetEntries() const -> std::span<const ConfigEntry>;

private:
    std::vector<ConfigEntry> m_entries;
    std::unordered_map<std::string, toml::table> m_tableCache;
};
```

**LoadTable implementation (in header or separate .cpp):**
- Check `m_tableCache` for the key. If found, return reference to cached table.
- Otherwise: load Layer 2 (`configDir / "{key}.toml"`). If file exists, `toml::parse_file()` wrapped in try/catch. If file doesn't exist, `toml::table{}` (empty) and log Info: "No config/{key}.toml found, using defaults".
- Load Layer 3 (`savedDir / "config" / "{key}.toml"`). If file exists, merge into Layer 2 table using shallow `insert_or_assign` per key. Log Info: "Applied user overrides from saved/config/{key}.toml".
- Cache merged table in `m_tableCache`. Return reference.
- On parse error: return `MakeError(std::format("Failed to parse config/{}.toml: {}", key, ex.what()))`.
- Includes: `<toml++/toml.hpp>` (or whatever the project's include path is - check existing EngineConfig.cpp for the correct include).

**2. Create `engine/wayfinder/src/app/ConfigService.h`** in `namespace Wayfinder`:

ConfigService is an AppSubsystem that provides address-stable config storage at runtime.

```cpp
class ConfigService : public AppSubsystem
{
public:
    /// Retrieve a config struct by type. Asserts if not loaded.
    template<typename T>
    [[nodiscard]] auto Get() const -> const T&;

    /// Retrieve a config struct by type, or nullptr if not loaded.
    template<typename T>
    [[nodiscard]] auto TryGet() const -> const T*;

    /// Store a config value. Called during AppBuilder setup (before Initialise).
    template<typename T>
    void Store(T config);

    // AppSubsystem lifecycle
    [[nodiscard]] auto Initialise(EngineContext& context) -> Result<void> override;
    void Shutdown() override;

private:
    struct ConfigDeleter
    {
        void (*Destroy)(void*) = nullptr;
        void operator()(void* ptr) const { if (Destroy and ptr) Destroy(ptr); }
    };

    std::unordered_map<std::type_index,
        std::unique_ptr<void, ConfigDeleter>> m_configs;
};
```

Implementation of Store<T>:
- `auto* raw = new T(std::move(config));`
- `auto deleter = ConfigDeleter{[](void* p) { delete static_cast<T*>(p); }};`
- `m_configs.emplace(std::type_index(typeid(T)), std::unique_ptr<void, ConfigDeleter>(raw, deleter));`

Implementation of Get<T>: lookup by type_index, assert if not found, return `*static_cast<const T*>(it->second.get())`.
Implementation of TryGet<T>: lookup, return nullptr or cast pointer.
Initialise: return `{}` (configs already stored during build phase).
Shutdown: `m_configs.clear()`.

**3. Add `LoadConfig<T>(key)` to `AppBuilder.h`:**

```cpp
/// Load a per-plugin config struct from TOML.
/// Per D-06: 3-tier layering (struct defaults < config/<key>.toml < saved/config/<key>.toml).
/// Per D-07: Cached - multiple calls with same key parse the file once.
/// Per D-09: Missing files return T{} (struct defaults) with Info log.
/// The loaded config is available via ConfigService::Get<T>() after Finalise.
template<typename T>
auto LoadConfig(std::string_view key) -> T
{
    auto& configReg = Registrar<ConfigRegistrar>();
    configReg.DeclareConfig(key, std::type_index(typeid(T)));

    // If no project paths set, return defaults
    if (m_configDir.empty())
    {
        return T{};
    }

    auto tableResult = configReg.LoadTable(key, m_configDir, m_savedDir);
    if (not tableResult)
    {
        // Parse error - log warning, return defaults
        Wayfinder::Log::Warn("Config", "Config load failed for '{}': {}", key, tableResult.error().GetMessage());
        return T{};
    }

    // Deserialise from TOML table. T must provide static FromToml(const toml::table&) -> T.
    return T::FromToml(*tableResult);
}
```
