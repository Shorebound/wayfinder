**1. Create `engine/wayfinder/src/plugins/IRegistrar.h`** in `namespace Wayfinder`:

```cpp
class IRegistrar
{
public:
    virtual ~IRegistrar() = default;
};
```

This is the base for type-erased registrar storage in AppBuilder. Concrete registrars (State, System, Tag, Lifecycle, Config, Subsystem) derive from this. No `Finalise()` on the base - each concrete registrar defines its own typed `Finalise() -> Result<OutputType>`. The base only provides a virtual destructor for `unique_ptr<IRegistrar>` cleanup.

Note: A `RegistrarBase<TOutput>` CRTP base with `virtual Finalise() -> Result<TOutput>` would be ideal but is not possible because AppBuilder needs to call Finalise() through IRegistrar* without knowing TOutput. Instead, AppBuilder will call Finalise() on each concrete registrar type directly (it knows the types at registration time). The IRegistrar base is purely for storage.

**2. Create `engine/wayfinder/src/app/AppDescriptor.h`** in `namespace Wayfinder`:

Per D-12, D-14: AppDescriptor holds processed outputs from registrar Finalise() calls. Type-erased storage queryable by output type.

```cpp
class AppDescriptor
{
public:
    AppDescriptor() = default;

    // Move-only (no copy)
    AppDescriptor(const AppDescriptor&) = delete;
    auto operator=(const AppDescriptor&) -> AppDescriptor& = delete;
    AppDescriptor(AppDescriptor&&) noexcept = default;
    auto operator=(AppDescriptor&&) noexcept -> AppDescriptor& = default;

    /// Retrieve a processed output by type. Asserts if not found.
    template<typename TOutput>
    [[nodiscard]] auto Get() const -> const TOutput&;

    /// Retrieve a processed output by type, or nullptr if not found.
    template<typename TOutput>
    [[nodiscard]] auto TryGet() const -> const TOutput*;

    /// Check if an output type is present.
    template<typename TOutput>
    [[nodiscard]] auto Has() const -> bool;

private:
    friend class AppBuilder;  // only AppBuilder can add outputs

    /// Add a processed output. Called by AppBuilder::Finalise().
    template<typename TOutput>
    void AddOutput(TOutput output);

    struct OutputDeleter
    {
        void (*Destroy)(void*) = nullptr;
        void operator()(void* ptr) const { if (Destroy and ptr) Destroy(ptr); }
    };

    std::unordered_map<std::type_index,
        std::unique_ptr<void, OutputDeleter>> m_outputs;
};
```

Implementation of `AddOutput<TOutput>`:
- Creates `new TOutput(std::move(output))` on the heap.
- Stores with a deleter that calls `delete static_cast<TOutput*>(ptr)`.
- Key is `std::type_index(typeid(TOutput))`.

Implementation of `Get<TOutput>`:
- Looks up `std::type_index(typeid(TOutput))` in `m_outputs`.
- Asserts if not found (using `WAYFINDER_ASSERT`).
- Returns `*static_cast<const TOutput*>(it->second.get())`.

Implementation of `TryGet<TOutput>`:
- Looks up, returns nullptr if not found.
- Returns `static_cast<const TOutput*>(it->second.get())`.

Includes: `<memory>`, `<typeindex>`, `<unordered_map>`, `core/Assert.h`.
