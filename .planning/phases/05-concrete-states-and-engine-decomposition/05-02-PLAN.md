  <action>
Create five AppSubsystem wrapper classes per D-05, D-06, D-07, D-08:

**WindowSubsystem** (app/WindowSubsystem.h + .cpp):
- Inherits AppSubsystem. Owns `std::unique_ptr<Window>`.
- `Initialise(EngineContext&)`: Creates Window via `Window::Create()` using config from context. Returns `Result<void>`.
- `Shutdown()`: Calls `m_window->Shutdown()`, resets unique_ptr.
- Accessors: `auto GetWindow() -> Window&`, `auto ShouldClose() const -> bool`.
- The init logic mirrors what EngineRuntime::Initialise() does for window creation.

**InputSubsystem** (app/InputSubsystem.h + .cpp):
- Inherits AppSubsystem. Owns `std::unique_ptr<Input>`.
- `Initialise(EngineContext&)`: Creates appropriate Input backend. Returns `Result<void>`.
- `Shutdown()`: Resets unique_ptr.
- Accessor: `auto GetInput() -> Input&`.

**TimeSubsystem** (app/TimeSubsystem.h + .cpp):
- Inherits AppSubsystem. Owns `std::unique_ptr<Time>`.
- `Initialise(EngineContext&)`: Creates Time. Returns `Result<void>`.
- `Shutdown()`: Resets unique_ptr.
- Accessors: `auto GetTime() -> Time&`, `auto GetDeltaTime() const -> float`.

**RenderDeviceSubsystem** (app/RenderDeviceSubsystem.h + .cpp):
- Inherits AppSubsystem. Owns `std::unique_ptr<RenderDevice>`.
- `Initialise(EngineContext&)`: Creates RenderDevice, initialises it using `context.GetAppSubsystem<WindowSubsystem>().GetWindow()`. Returns `Result<void>`.
- `Shutdown()`: Shuts down and resets device.
- Accessor: `auto GetDevice() -> RenderDevice&`.

**RendererSubsystem** (app/RendererSubsystem.h + .cpp):
- Inherits AppSubsystem. Owns `std::unique_ptr<Renderer>`, `FrameCanvases`, and `BlendableEffectRegistry`.
- `Initialise(EngineContext&)`: Creates Renderer using device from `context.GetAppSubsystem<RenderDeviceSubsystem>().GetDevice()`. Sets up BlendableEffectRegistry. Returns `Result<void>`.
- `Shutdown()`: Shuts down renderer, resets unique_ptr.
- Accessors: `auto GetRenderer() -> Renderer&`, `auto GetCanvases() -> FrameCanvases&` (and const overload), `auto GetBlendableEffectRegistry() -> BlendableEffectRegistry&` (and const overload).
- Per D-07: No global SetActiveInstance pattern. Registry accessed via EngineContext -> RendererSubsystem -> GetBlendableEffectRegistry().

All subsystems use `Wayfinder::Log` for init/shutdown logging.

Add all 10 new files (5 .h + 5 .cpp) to `engine/wayfinder/CMakeLists.txt` in the App section, maintaining alphabetical order.

**CRITICAL**: These subsystems coexist with EngineRuntime. They do NOT replace it yet (Phase 6). No changes to EngineRuntime or Application in this plan.
